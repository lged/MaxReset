# 3dsMax 恢复出厂设置

![QQ截图20191203092201](https://user-images.githubusercontent.com/19682422/70013101-797f4500-15b1-11ea-88bb-22a1353705aa.png)

### 用途
- 将 3dsMax 的设置恢复到初始状态

- 解决 Max 出现的各种疑难杂症

- 清除某些 Max 病毒

### 原理
- 将 3dsMax 的配置文件删除，重启 Max 后便会自动生成默认的配置文件


## 获取源码
    
    git clone https://github.com/lged/MaxReset.git
